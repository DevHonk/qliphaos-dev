## Run instructions

## Linux

**System Requirements**
* Qemu / VirtualBox

#### Ubuntu / Debian

**Install Dependencies**

Qemu

```bash
$ sudo apt-get install qemu-system-i386 qemu-utils
```

---

### Run

> Note: The project must be compiled before it can run.
> [Build instructions]()

**Qemu**
Go into the `Toolchain/` directory and run the run.sh script.

```bash
$ ./run.sh qemu
```
